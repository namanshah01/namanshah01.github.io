APIWink at core is an API micro-lending service, ideal to attract more users and reduce costs. Traditionally monthly/yearly API subscription models are costly for devs experimenting APIs for their platform. This micro-lending service enables purchasing "API Count" for Skale programmed currency. This pay per use model offers flexibility to user, exposure to provider while being revenue positive for them and APIWink platform.
